## 📖 About This Repository

This repository documents my journey of learning **Mathematics and Python simultaneously**.

Rather than studying mathematical concepts theoretically, I implement them in Python to develop a deeper understanding of both the mathematics and the programming logic behind them.

Every program in this repository is designed to answer a simple question:

> **"Can this mathematical concept be understood better through code?"**

The answer is explored through practical implementations, experimentation, and continuous improvement.

## 💡 Learning Methodology

For every concept:

1. Study the mathematical idea.
2. Understand the underlying logic.
3. Convert the logic into an algorithm.
4. Implement it in Python.
5. Test different cases.
6. Document key learnings.

This process transforms formulas into intuition and theory into practical understanding.
